# isite

Convert **i**ssues to a web**site**.

## Usage

```bash
isite generate --help
```

After generating the Markdown based documents, you can build the website with the following engines.

## Engines

- [x] [zola](https://github.com/getzola/zola)
- [ ] [hugo](https://github.com/gohugoio/hugo)
